# Licensing and provenance

Wikidata structured data is CC0. Every accepted Commons file has a separate license record.

Webwoven's project-authored synthetic smoke-fixture data is distributed under the repository's
MIT license. It is test-only data, not imported Wikidata knowledge, and normal gameplay never
silently falls back to it.

The release playtest pack contains real Wikidata labels, descriptions, and statements under CC0.
Its attribution report records the source and snapshot date. Accepted Commons derivatives are
stored inside the immutable graph bundle; runtime gameplay never fetches them from Wikimedia.

The Build Week automatic media allowlist is Public Domain, CC0, and CC BY 4.0 with complete source
and creator fields. CC BY-SA, unclear, multi-license, restricted, and incomplete records are not
published automatically.

Every accepted asset record stores source page, original URL, creator/credit, license and URL,
remote hash, local hash, retrieval time, modifications, and automatic review status. Media is
published only when its entity link, raster type, HTTPS Wikimedia origin, complete attribution,
license allowlist, content hash, and manifest entry all pass. A project-authored category
illustration replaces missing or rejected endpoint media.

The July 15 release pack localizes 75 curated endpoint images: 65 Public Domain, three CC0, and
seven CC BY 4.0. Seventy-nine candidates fail metadata or license policy and one animated GIF
fails the static-raster policy; none of those files enters the bundle.
