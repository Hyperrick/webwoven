# Licensing and provenance

Wikidata structured data is CC0. Every accepted Commons file has a separate license record.

Webwoven's project-authored synthetic smoke-fixture data is distributed under the repository's
MIT license. It is test-only data, not imported Wikidata knowledge, and normal gameplay never
silently falls back to it.

The July 13 local playtest pack contains real Wikidata labels, descriptions, and statements under
CC0. Its attribution report records the source and snapshot date. It intentionally contains no
Commons files yet; the category artwork is project-authored and labelled as a non-documentary
fallback.

The Build Week automatic media allowlist is Public Domain, CC0, and CC BY 4.0 with complete source
and creator fields. CC BY-SA, unclear, multi-license, restricted, and incomplete records are not
published automatically.

Every approved asset record stores source page, original URL, creator/credit, license and URL,
remote hash, local hash, retrieval time, modifications, and review status. Start and target media
must be manually approved. A category illustration replaces missing intermediate media.
