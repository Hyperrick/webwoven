# Contributing

Webwoven is being built in public during OpenAI Build Week.

1. Read `AGENTS.md` and the architecture documentation.
2. Keep changes focused and modular; authored code files stay under 600 LOC.
3. Add tests for behavioral changes.
4. Update the relevant documentation and current build log.
5. Never commit credentials, raw production snapshots, or unreviewed licensed media.
6. Run the complete quality gate before opening a pull request.

Use `agent/<short-description>` for Codex-authored branches. Pull requests should explain the
change, its product impact, the validation performed, and any data or license implications.
