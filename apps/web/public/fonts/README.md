# Typeface licensing

Webwoven self-hosts the following fonts through the pinned Fontsource packages in the web bundle:

- Newsreader Variable — SIL Open Font License 1.1
- IBM Plex Sans Variable — SIL Open Font License 1.1
- IBM Plex Mono 500 — SIL Open Font License 1.1

The distributed font files and complete license texts are supplied by their respective Fontsource packages. They are bundled by Vite and are never requested from a third-party font CDN at runtime.
